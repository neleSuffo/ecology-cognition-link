# Linking Preschoolers' Everyday Social Ecology to Language and Pragmatic Development

This repository contains the complete analytical pipeline, statistical models, and visualization code for the manuscript:  
**"Language & Pragmatics: How everyday experiences relate to developmental domains in the preschool years"**.

## Overview

We combine Bayesian linear variable selection with gradient-boosted decision tree architectures to examine how preschoolers' everyday social environments (vocal activity, speech exposure, turn-taking dynamics, and visual interaction) predict receptive vocabulary (**oREV**) and pragmatic abilities (**PRAG**). 

Our dual modeling framework captures both linear bounds and non-linear, non-monotonic feature interactions using:
1. **Linear Bayesian Inference:** Reference modeling using `brms` paired with decision-theoretic forward variable selection via `projpred`.
2. **Nonlinear Gradient Boosting:** Gradient-boosted decision trees using `CatBoost` evaluated via single-loop Leave-One-Out (LOO) cross-validation, interpreted using Out-of-Fold (OOF) SHAP values (*SHapley Additive exPlanations*).
